What is CSS:-

1. CSS stands for Cascading Style Sheet(CSS).
2. CSS basically used to design the html webpages/documents.
3. The Current version of css is CSS3.
4. CSS was developed by Haykon Willium in 1994.

Types Of CSS:-

** There are mainly three types of CSS present, these are:

1. Inline CSS: used inside any html tag to design that specific element.
* Syntax:
OpentagName style="property: value">Content ClosetagName

2. Internal CSS: used inside any html document to design that single html document.
* Syntax:
openStyleTag
propertyName : value
closeStyleTag

3. External CSS: can be used to design multiple html documents. We need to link that external file in the html document to design. Link tag is used to link that css file.
* Syntax: linkTag rel = "style.css" type="styleSheet"
